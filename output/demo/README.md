# BidRadar-X 演示视频交付

- `BidRadar-X-Demo-zh-CN.mp4`：1920×1080、H.264 + AAC、中文男声、原创电子氛围音。
- `BidRadar-X-Demo-zh-CN.srt`：可导入剪映、Premiere、Final Cut 或播放器的中文字幕。
- `BidRadar-X-Demo-旁白稿.md`：完整中文旁白，便于人工复录或答辩修改。
- `tools/build_demo_video.py`：仓库根目录下的可复现构建脚本。

仓库只保留面向评委的最终视频、封面、字幕和旁白稿。音频分段、逐镜图片、原始视频和质检抽帧属于本地构建中间产物，已通过 `.gitignore` 排除，避免无意义地增大仓库。

## 内容边界

- 账户授权型来源以“独立适配器、凭据不公开”呈现，不在视频中暴露 Token 或 API Key。
- 飞书企业知识库归档明确标注为“即将上线、需企业凭据授权”，没有冒充已经完成的线上联调。
- AI 辅助摘要与风险研判受原始公告证据约束，不宣称模型可以替代原文事实。
